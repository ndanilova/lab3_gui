# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ah/2
- Trapezoid: S = h(a + b)/2
- Rectangle: S = ab

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c
- Trapezoid: P = a + b + c + d
- Rectangle: P = 2(a + b)
